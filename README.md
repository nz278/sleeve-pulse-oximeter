# Pulse Oximeter Prototype

An Arduino-based wearable sleeve pulse oximeter prototype that measures 
heart rate and blood oxygen saturation (SpO₂) levels, transmits readings 
over Bluetooth (BLE), and supports vibration alerts. This project was 
developed as an educational embedded systems project for continuous health
monitoring as part of the MIT Beaver Works Summer Institute E-Textiles 
and Wearable Technology program.

## Features

-   Measure heart rate (BPM)
-   Measure blood oxygen saturation (SpO₂)
-   Transmit readings via Bluetooth (BLE)
-   Serial Monitor output for debugging
-   Individual hardware test sketches

## Software Requirements

-   Arduino IDE
-   SparkFun Bio Sensor Hub library
-   Adafruit Bluefruit LE library
